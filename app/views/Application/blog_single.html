    有人还记得我 600 多天的轮子吗 ( http://v2ex.com/t/140281)，这次我又给它造了个轮子。

    不管是否优雅，不过这的确是个 Markdown 编辑器，它支持:

    • GFM 语法
    • Math 语法
    • Vim 模式
    • 无干扰模式
    • 导出 PDF 图片等等
    • 多个 tab 支持
    • ...

    基于 Electron 和 Vue 开发，你可以来帮助它支持更多功能。

    关于 Electron 有很多人不喜欢，理由是太大、对电池不好等等，我可以提供一个解决方案，就是忽略这个帖子。
